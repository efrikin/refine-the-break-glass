# `Refine the Break-Glass`

## `Usage`

```bash
git clone https://github.com/efrikin/refine-the-break-glass.git
git lfs pull
```

### `Preview mode`

#### `Up`

```bash
podman kube play quarto-preview.yaml
```

#### `Down`

```bash
podman kube play quarto-preview.yaml --down
```

#### `Debugging`

```bash
podman pod logs quarto-preview
```
